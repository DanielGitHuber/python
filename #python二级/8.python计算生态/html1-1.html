苏拉皱盔犀鸟

摄影：Tim Laman, Nat Geo Image Collection
 

潜水者冲着调节器激动地尖叫，兴奋得手舞足蹈。那是2020年8月，在佛罗里达群岛水下4米处，莫特海洋实验室和水族馆的海洋生物学家汉娜·科赫及其同事一直在一处暗礁上方游弋等待。就在午夜降临之前，暗礁周围的珊瑚发出无声的爆裂，一束束细小的粉色与橙色精子和卵子喷薄而出，开始向海面升腾，挥洒出一幅以新生命做笔墨的点彩派巨作。

潜水小组的快乐之舞让周遭的海洋发光生物冒出了电蓝色的光芒。“ 看起来就像是我们自己创造的烟花秀，” 科赫说，“还挺美。”

这种突然的爆发是很多造礁珊瑚的典型繁殖方式——通常每年一次，某个满月后数日的一个夏夜。受月球周期、水温和日照时间的影响，佛罗里达珊瑚礁上的不同珊瑚物种同时释放出数万亿精子和数百万个卵子。这是一场促进遗传多样性的狂欢，能确保小部分卵子成功受精，长成幼虫定居礁石，并为下一代播种。

但这不是普通的产卵事件。2015年，这些被美国《濒危物种法》列为受威胁物种的山星珊瑚由莫特海洋实验室的科学家们进行了繁育和“种植”，作为珊瑚礁修复计划的一部分。它们在当年的珊瑚白化事件中得以幸存，又顽强地挺过了2017年的四级飓风以及两年后爆发的一场疫病，展现出令人振奋的适应力。它们比野生珊瑚更快进入繁殖成熟期，成为第一批得以恢复的成石珊瑚，在海上产卵繁育。

对于科学家们来说，这是一个可喜的里程碑，他们迫切抢救珊瑚免受气候变化和其他人为威胁带来的破坏性影响。在已知800多种造礁珊瑚中，超过1/4的物种被国际自然保护联盟列入脆弱、濒危或极度濒危名单，联盟发出警告，珊瑚灭绝的风险会随着温度升高而增加。
